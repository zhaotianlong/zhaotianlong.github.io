vue vs angular

2019-01-09

1.类型分离，angular 开发更关注的是 html css js 的拆分多文件组件,vue 更推崇单文件组件 2.语法上区别 :
2.1 绑定属性与绑定事件 vue :属性 angular [属性],vue @事件 angular (事件)
2.2 双向绑定:vue v-model=“属性” ,angular [ngModel]=[属性][(dom属性)] = “属性”
2.3 结构语句：v-if ngIf;v-for ngfor ngfor 不支持 in key => track-ny;v-show angular [class.hide];anular 不允许 ngfor 与 ng if 在一个节点上
2.4 生命周期：created => init ; mounted => contentinit,viewinit;destory = ;ngchange 约= update;
2.5 子组件区分：angular 一种投影插件 一种子组件；
2.6 属性：angular 一个 get set 解决，vue 计算属性 监听属性
2.7 动态类: :class = “{类名:属性}” :class=”[类名 1,类名 2]” angular [class]=”{类名:属性}”
2.8 动态模版：angular ng-templete ng-content => vue templete
2.9 插槽：slot -> ng-content
2.10 区分 vuex 与 angular redux 区分
2.11 组件通信

2.12 vue loader 这是 angular 没有的提过的
2.13 vue vuerouter router-view 与 angular router-outlet
2.14 vue vuex actions 与 effects 有区别 不能串行
